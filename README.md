# Phaser 3 + Webpack 4

A boilerplate project for start game development with Phaser 3 + Webpack 4

## Install and run

| Command              | Description                                            |
| -------------------- | ------------------------------------------------------ |
| `npm install`        | Install dependencies                                   |
| `npm start`          | Build app and launch browser                           |
| `npm run build`      | Build production ready app                             |
