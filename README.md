# Phaser 3 + Webpack 4

Start game development with Phaser + Webpack.

## Install and run

| Command              | Description                                            |
| -------------------- | ------------------------------------------------------ |
| `npm install`        | Install dependencies                                   |
| `npm start`          | Build app and launch browser                           |
